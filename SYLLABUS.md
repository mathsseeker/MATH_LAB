# Mathematics Laboratory for Industrial Engineering
## Course Syllabus

### Course Information
- **Course Code**: IE-MATH-LAB-101
- **Credits**: 3
- **Duration**: 15 weeks
- **Lab Hours**: 3 hours per week

### Course Description

This laboratory course provides practical experience with mathematical tools and computational methods essential for Industrial Engineering. Students will learn to implement and apply mathematical concepts using programming tools, focusing on real-world engineering problems.

### Course Objectives

1. Develop proficiency in using computational tools for mathematical problem-solving
2. Apply linear algebra techniques to engineering systems
3. Implement optimization algorithms for industrial applications
4. Perform statistical analysis and probability calculations
5. Use numerical methods for solving engineering problems
6. Model and simulate industrial processes

### Weekly Schedule

#### Week 1-2: Introduction to Mathematical Computing
- Introduction to Python/MATLAB for mathematical computing
- Arrays, matrices, and basic operations
- Visualization and plotting

#### Week 3-4: Linear Algebra
- Matrix operations and properties
- Solving systems of linear equations
- Eigenvalues and eigenvectors
- Applications in engineering systems

#### Week 5-6: Optimization
- Linear programming
- Nonlinear optimization
- Constrained and unconstrained optimization
- Applications in resource allocation and scheduling

#### Week 7-8: Probability and Statistics
- Probability distributions
- Statistical inference
- Hypothesis testing
- Regression analysis
- Quality control applications

#### Week 9-10: Numerical Methods
- Root finding algorithms
- Numerical integration and differentiation
- Solving differential equations
- Error analysis

#### Week 11-12: Operations Research
- Queueing theory
- Inventory models
- Network optimization
- Decision analysis

#### Week 13-14: Simulation and Modeling
- Monte Carlo simulation
- Discrete event simulation
- System dynamics
- Production system modeling

#### Week 15: Final Project
- Comprehensive project integrating multiple concepts
- Presentation and documentation

### Required Software

- Python 3.8+ with packages:
  - NumPy
  - SciPy
  - Matplotlib
  - Pandas
  - Jupyter Notebook
- Alternative: MATLAB R2020a or later

### Course Learning Outcomes

Upon successful completion of this course, students will be able to:

1. **Apply** matrix operations to model and solve engineering systems
2. **Implement** optimization algorithms for industrial problems
3. **Analyze** data using statistical methods and probability theory
4. **Develop** numerical solutions for mathematical problems
5. **Model** complex industrial systems using simulation
6. **Evaluate** solution quality and computational efficiency
7. **Communicate** technical results effectively through documentation
