# Laboratory Exercises

This directory contains hands-on laboratory exercises for the Math Lab course.

## Available Labs

1. **lab1_intro_to_python.md** - Introduction to Python for Mathematical Computing
   - NumPy basics
   - Matrix operations
   - Array indexing
   - Mathematical functions and visualization

2. **lab2_linear_algebra.md** - Linear Algebra Applications in Industrial Engineering
   - Production planning
   - Network flow analysis
   - Quality control (PCA)
   - Markov chains

3. **lab3_optimization.md** - Optimization Techniques for Industrial Engineering
   - Linear programming
   - Transportation problems
   - EOQ (Economic Order Quantity)
   - Multi-objective optimization

## How to Use

1. Read the lab document thoroughly
2. Review the objectives and background
3. Copy the starter code
4. Complete each exercise
5. Test your solutions
6. Submit according to guidelines

## Submission

- Create a Python file: `labN_yourname.py`
- Include all required code and comments
- Generate required visualizations
- Submit via GitHub repository

## Getting Help

- Review course examples in `/examples`
- Check the quick reference in `/resources`
- Consult documentation for NumPy, SciPy, Matplotlib
- Create an issue for questions

## Tips

- Start early
- Test incrementally
- Verify results with simple cases
- Comment your code
- Format output clearly
