# Code Examples

This directory contains working code examples demonstrating key mathematical concepts used in Industrial Engineering.

## Available Examples

### example1_matrix_operations.py
Demonstrates matrix operations commonly used in IE:
- Resource allocation matrices
- Cost calculations
- Solving linear systems
- Eigenvalue analysis for quality control
- Markov chains for machine state analysis
- Linear transformations visualization

**Run with:**
```bash
python example1_matrix_operations.py
```

### example2_optimization.py
Demonstrates optimization techniques:
- Linear programming for production planning
- Economic Order Quantity (EOQ) optimization
- Constrained nonlinear optimization
- Sensitivity analysis

**Run with:**
```bash
python example2_optimization.py
```
